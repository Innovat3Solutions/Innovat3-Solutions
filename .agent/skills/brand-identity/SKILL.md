# Innovate Solutions Brand Identity

## When to use this skill
- When writing CSS or styling components for Innovate Solutions.
- When generating images or UI mockups.
- When creating the portfolio website plan.

## Brand Assets

### Colors
| Token | Hex | Usage |
| :--- | :--- | :--- |
| **Primary** | `#84CC16` | Main actions, highlights (Lime Green) |
| **Accent** | `#94A3B8` | Subtitles, borders, secondary elements (Slate Grey) |
| **Background** | `#FFFFFF` | Page background (White - Light Mode) |
| **Text Primary** | `#000000` | Headings, body text |
| **Link** | `#94A3B8` | Anchor tags |

### Typography
- **Primary Font**: `Inter` (Sans-serif)
- **Headings**: `Inter` (Bold/ExtraBold)
- **Body**: `Inter` (Regular)
- **Sizes**:
    - H1: `64px`
    - H2: `40px`
    - Body: `14px`

### Graphics
- **Favicon**: `https://storage.googleapis.com/msgsndr/1VKsWhjzYeGOHMwxwbBN/media/67ffb093c38e3ea5a71bd11f.png`
- **OG Image**: `https://assets.cdn.filesafe.space/1VKsWhjzYeGOHMwxwbBN/media/692099442965fa445fc60c65.jpg`

## Design System Rules
1.  **Rounding**: Use `border-radius: 4px` for buttons and cards.
2.  **Spacing**: Base unit is `10px`. Layouts should be spacious.
3.  **Vibe**: Clean, professional, modern, "Innovate" theme.

## Instructions
- ALWAYS use variables for colors (e.g., `var(--color-primary)`).
- ALWAYS import the Inter font from Google Fonts.
- When asked to "design" or "style", refer to this table first.
